# European Dynamics Submission Test

A simple full-stack web application for user management built with Spring Boot and Angular.

## Features

- **Register new users** via a simple form (name, surname, gender, address).
- **View all users** in a clean, searchable table.
- **Edit user details** easily update user information.
- **Delete users** remove users quickly from the system.
- **Smooth navigation** with a modern, user-friendly interface.

## Tech Stack

- **Backend**: Spring Boot REST API
- **Frontend**: Angular with Tailwind CSS
- **Database**: MySQL
- **Styling**: Clean, modern UI with Tailwind CSS

## Getting Started

### Backend Setup

1. Navigate to the backend directory:

```
cd ed-backend
```

2. Run the Spring Boot application:

```
mvn spring-boot:run
```

The backend API will start on `http://localhost:8080`

### Frontend Setup

1. Navigate to the frontend directory:

```
cd ed-frontend
```

2. Install dependencies:

```
npm install
```

3. Start the development server:

```
npm run start
```

The frontend application will be available at `http://localhost:4200`

## Pages

- **Home**: Landing page with overview
- **Register**: Form to add new users
- **Users**: View all registered users with edit and delete options
